# Tech stack

- Next.js 15 App Router & TypeScript: The foundational framework for the application, ensuring type safety and efficient routing.
- Prisma ORM + Neon Postgres: Used for the database layer, providing type-safe database queries.
- Better Auth: Handles authentication, offering various providers and streamlining complex authentication flows.
- Polar: Manages payments and subscriptions, including free tiers, paid plans, and usage-based billing.
- Inngest: Powers background job execution, workflow automation, and real-time updates through pub/sub messaging.
- React Flow: Provides the visual drag-and-drop canvas for building workflows.
- AI Integrations: Supports popular AI providers like OpenAI, Claude, and Gemini.
- Sentry: Integrated for error tracking, logging, session replays, and AI agent monitoring
- Code Rabbit: Used for AI-powered pull request reviews, facilitating a professional Git workflow
